# 倒计时页面

一个精美的倒计时页面，使用 Vue 3 + Element Plus 构建，支持自定义目标日期，具有现代化的 UI 设计和流畅的动画效果。

## 功能特性

- ⏰ 实时倒计时显示（天/时/分/秒）
- 📅 日期选择器，支持修改目标日期
- 💾 自动保存到 localStorage，刷新页面不丢失
- 🎨 精美的渐变背景和毛玻璃效果
- 📱 完全响应式设计，支持移动端和桌面端
- ✨ 流畅的动画效果和交互体验

## 技术栈

- **Vue 3** - 渐进式 JavaScript 框架
- **Element Plus** - 基于 Vue 3 的组件库
- **Vite** - 下一代前端构建工具
- **Day.js** - 轻量级日期处理库

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

开发服务器将在 `http://localhost:3000` 启动，并自动在浏览器中打开。

### 构建生产版本

```bash
npm run build
```

构建后的文件将输出到 `dist` 目录。

### 预览生产版本

```bash
npm run preview
```

## 使用说明

1. 打开页面后，默认显示距离30天后的倒计时
2. 点击日期选择器，选择你的目标日期
3. 选择的日期会自动保存到浏览器本地存储
4. 刷新页面后，之前选择的日期会自动恢复

## 设计特点

- **渐变背景**：使用紫色到粉色的渐变背景，视觉效果优雅
- **毛玻璃效果**：卡片使用半透明背景和模糊效果，现代感十足
- **响应式布局**：
  - 桌面端：4列网格显示时间单位
  - 平板/手机：2x2网格布局
- **动画效果**：
  - 数字变化有平滑过渡动画
  - 卡片 hover 时有缩放效果
  - 页面加载有淡入动画
